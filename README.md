# ARSII App

## Getting Started

This project consists of a front-end (Expo) and a back-end (Node.js) application.

### Prerequisites
- Node.js and npm installed
- Expo CLI installed (`npm install -g expo-cli`)

## Running the Application

### Front-End
To run the front-end application:
```bash
npx expo start
```

### Back-End
To run the back-end server:
```bash
npm start
```

## Project Structure
- **Front-End**: Built with React Native and Expo
- **Back-End**: Built with Node.js
